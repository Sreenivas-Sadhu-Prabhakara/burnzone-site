# Train

Your workouts and programs, ready to go.

## Workout library <span class="bz-badge live">Live</span>

Browse the gym's workout catalogue. Each workout shows:

- **Name** and a short description
- **Duration** (minutes) and **difficulty** (beginner → advanced)
- **Estimated calories** per session
- A **video link** to follow along

## Curated programs <span class="bz-badge live">Live</span>

Programs bundle workouts into a plan your gym's trainers have put together — for
example *Beginner Kickstart* or *Shred & Strength*. Each program shows its workouts,
status, and total estimated burn, so you can pick a path instead of guessing.

## Loading, empty & error states

Every screen handles the real world: a spinner while loading, a friendly empty state
when there's nothing yet, and a **retry** if the network hiccups. Pull to refresh any
time.
